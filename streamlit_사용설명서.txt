      엑셀 측정 데이터 입력 시스템 (Streamlit 웹 버전)
                     사용 설명서
═══════════════════════════════════════════════════════════

🌐 웹 인터페이스 사용법
───────────────────────────────────────────────────────────

1단계: 파일 업로드
  ↓ 왼쪽 사이드바에서 "Browse files" 클릭
  ↓ 엑셀 파일(.xlsx, .xlsm) 선택

2단계: 설정
  ↓ 시트 선택 (드롭다운 메뉴)
  ↓ 시작 열 입력 (기본: F)
  ↓ 시작 행 입력 (기본: 5)

3단계: 데이터 입력
  ↓ 입력란에 측정 데이터 입력
  ↓ 한 줄에 하나의 항목
  ↓ 쉼표(,)로 구분

4단계: 처리
  ↓ "데이터 처리" 버튼 클릭
  ↓ 처리 결과 확인

5단계: 다운로드
  ↓ "결과 파일 다운로드" 버튼 클릭
  ↓ 브라우저 다운로드 폴더에 저장


📊 입력 형식 (CLI 버전과 동일)
───────────────────────────────────────────────────────────

1️⃣ 단순 측정값
   형식: 항목번호, 행개수, 기준값, 상한공차, 하한공차
   예시: 51, 1, 7.0, 0.15, 0.15
   결과: 7.0, 0.15, -0.15, 6.85, 7.15

2️⃣ 위치도 값 (기준값에 Ø 포함)
   형식: 항목번호, 행개수, 기준값, 상한공차, 하한공차
   예시: 55, 4, Ø4.25, 0.15, 0.15

3️⃣ 참고 값 (기준값을 괄호로 감싸기)
   형식: 항목번호, 행개수, (기준값), [상한공차], [하한공차]
   예시: 60, 1, (1.2)
   예시: 61, 3, (7.0), 0.15, 0.15, 참고

4️⃣ MMC 공차 (MMC 값에 'm' 붙이기)
   형식: 항목번호, 세트개수, MMC공차, [MAX값]
   예시: 70, 10, 0.2m
   예시: 70, 10, 0.2m, 0.5


🚀 일괄 입력 예시
───────────────────────────────────────────────────────────

입력란에 다음과 같이 입력:

51, 1, 7.0, 0.15, 0.15
52, 3, 10.5, 0.2, 0.1
55, 4, Ø4.25, 0.15, 0.15
60, 1, (1.2)
61, 3, (7.0), 0.15, 0.15, 참고
70, 10, 0.2m, 0.5

→ "데이터 처리" 버튼 클릭
→ 모든 항목이 자동으로 처리됩니다!


💡 웹 버전의 장점
───────────────────────────────────────────────────────────

✅ 직관적인 GUI 인터페이스
✅ 실시간 결과 미리보기
✅ 드래그 앤 드롭 파일 업로드
✅ 처리 결과 즉시 확인
✅ 원본 파일 수정 없이 새 파일 다운로드
✅ 여러 번 시도 가능
✅ 입력 형식 가이드 내장


⚙️ 고급 기능
───────────────────────────────────────────────────────────

📖 입력 형식 가이드 보기
  → 앱 상단의 "입력 형식 가이드" 확장 메뉴 클릭
  → 모든 입력 형식과 예시 확인

🔄 여러 번 처리
  → 결과가 마음에 들지 않으면 입력 수정 후 재처리
  → "입력 초기화" 버튼으로 처음부터 다시 시작

📋 처리 결과 확인
  → "처리 결과" 확장 메뉴에서 상세 내역 확인
  → 성공/경고/오류 메시지 구분 표시


🔧 문제 해결
───────────────────────────────────────────────────────────

Q: 웹 브라우저가 열리지 않아요
A: 수동으로 브라우저를 열고 http://localhost:8501 입력

Q: "Address already in use" 오류
A: 이미 실행 중인 앱이 있습니다
   → 기존 창을 닫거나 Ctrl+C로 종료 후 재실행

Q: 파일 업로드가 안 돼요
A: 파일 형식 확인 (.xlsx 또는 .xlsm만 가능)
   → 파일이 다른 프로그램에서 열려있지 않은지 확인

Q: "데이터 처리" 버튼이 비활성화되어 있어요
A: 데이터 입력란에 내용을 입력한 후 버튼을 클릭하세요

Q: 다운로드한 파일이 어디에 있나요?
A: 브라우저의 기본 다운로드 폴더를 확인하세요
   → 보통 "다운로드" 또는 "Downloads" 폴더

Q: 프로그램을 종료하려면?
A: 브라우저 창을 닫고, 명령 프롬프트 창에서 Ctrl+C


🌟 사용 팁
───────────────────────────────────────────────────────────

💾 원본 파일 보호
  → 웹 버전은 원본 파일을 수정하지 않습니다
  → 처리된 결과는 새 파일로 다운로드됩니다
  → 안전하게 여러 번 시도 가능!

📝 입력 템플릿 저장
  → 자주 사용하는 입력 패턴을 메모장에 저장
  → 복사-붙여넣기로 빠르게 입력

🎯 단계별 처리
  → 한 번에 많은 항목보다는
  → 5-10개씩 나눠서 처리하면 더 안전

🔍 결과 확인
  → 다운로드 전에 "처리 결과" 메뉴에서
  → 모든 항목이 제대로 처리되었는지 확인


📊 CLI vs 웹 버전 비교
───────────────────────────────────────────────────────────

CLI 버전 (excel_automation.py):
  ✅ 빠른 실행
  ✅ 파일 직접 수정
  ✅ 반복 작업에 적합
  ❌ 명령 프롬프트 사용 필요

웹 버전 (streamlit_app.py):
  ✅ 사용하기 쉬움
  ✅ 시각적 인터페이스
  ✅ 결과 미리보기
  ✅ 원본 파일 보호
  ❌ 초기 로딩 시간 필요

→ 둘 다 기능은 동일합니다!
→ 편한 방법을 선택하세요!
