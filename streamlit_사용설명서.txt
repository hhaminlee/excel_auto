      엑셀 측정 데이터 입력 시스템 (Streamlit 웹 버전)
                     사용 설명서
═══════════════════════════════════════════════════════════

🌐 웹 인터페이스 사용법
───────────────────────────────────────────────────────────

1단계: 파일 업로드
  ↓ 왼쪽 사이드바에서 "Browse files" 클릭
  ↓ 엑셀 파일(.xlsx, .xlsm) 선택

2단계: 설정
  ↓ 시트 선택 (드롭다운 메뉴)
  ↓ 시작 열 입력 (기본: F)
  ↓ 시작 행 입력 (기본: 5)

3단계: 데이터 입력
  ↓ 입력란에 측정 데이터 입력
  ↓ 한 줄에 하나의 항목
  ↓ 쉼표(,)로 구분

4단계: 처리
  ↓ "데이터 처리" 버튼 클릭
  ↓ 처리 결과 확인

5단계: 다운로드
  ↓ "결과 파일 다운로드" 버튼 클릭
  ↓ 브라우저 다운로드 폴더에 저장


📊 입력 형식 (CLI 버전과 동일)
───────────────────────────────────────────────────────────

1️⃣ 단순 측정값
   형식: 항목번호, 행개수, 기준값, 상한공차, 하한공차
   예시: 51, 1, 7.0, 0.15, 0.15
   결과: 7.0, 0.15, -0.15, 6.85, 7.15

2️⃣ 위치도 값 (기준값에 Ø 포함)
   형식: 항목번호, 행개수, 기준값, 상한공차, 하한공차
   예시: 55, 4, Ø4.25, 0.15, 0.15

3️⃣ 참고 값 (기준값을 괄호로 감싸기)
   형식: 항목번호, 행개수, (기준값), [상한공차], [하한공차]
   예시: 60, 1, (1.2)
   예시: 61, 3, (7.0), 0.15, 0.15, 참고

4️⃣ MMC 공차 (MMC 값에 'm' 붙이기)
   형식: 항목번호, 세트개수, MMC공차, [MAX값]
   예시: 70, 10, 0.2m
   예시: 70, 10, 0.2m, 0.5


🚀 일괄 입력 예시
───────────────────────────────────────────────────────────

입력란에 다음과 같이 입력:

51, 1, 7.0, 0.15, 0.15
52, 3, 10.5, 0.2, 0.1
55, 4, Ø4.25, 0.15, 0.15
60, 1, (1.2)
61, 3, (7.0), 0.15, 0.15, 참고
70, 10, 0.2m, 0.5

→ "데이터 처리" 버튼 클릭
→ 모든 항목이 자동으로 처리됩니다!


⚙️ 고급 기능
───────────────────────────────────────────────────────────

📖 입력 형식 가이드 보기
  → 앱 상단의 "입력 형식 가이드" 확장 메뉴 클릭
  → 모든 입력 형식과 예시 확인

🔄 여러 번 처리
  → 결과가 마음에 들지 않으면 입력 수정 후 재처리
  → "입력 초기화" 버튼으로 처음부터 다시 시작

📋 처리 결과 확인
  → "처리 결과" 확장 메뉴에서 상세 내역 확인
  → 성공/경고/오류 메시지 구분 표시


🌟 사용 팁
───────────────────────────────────────────────────────────

💾 원본 파일 보호
  → 웹 버전은 원본 파일을 수정하지 않습니다
  → 처리된 결과는 새 파일로 다운로드됩니다
  → 안전하게 여러 번 시도 가능!

📝 입력 템플릿 저장
  → 자주 사용하는 입력 패턴을 메모장에 저장
  → 복사-붙여넣기로 빠르게 입력

🎯 단계별 처리
  → 한 번에 많은 항목보다는
  → 5-10개씩 나눠서 처리하면 더 안전

🔍 결과 확인
  → 다운로드 전에 "처리 결과" 메뉴에서
  → 모든 항목이 제대로 처리되었는지 확인
